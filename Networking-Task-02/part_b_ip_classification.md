# Part B — IP Address Classification

**Submitted by:** Sanjana
**Date:** 06 June 2026

---

| IP Address | Category | Reason |
|------------|----------|--------|
| `192.168.1.10` | Private | Falls in the 192.168.0.0 – 192.168.255.255 private range |
| `10.0.0.5` | Private | Falls in the 10.0.0.0 – 10.255.255.255 private range |
| `172.16.5.20` | Private | Falls in the 172.16.0.0 – 172.31.255.255 private range |
| `8.8.8.8` | Public | Outside all private ranges — this is Google's public DNS server |
| `1.1.1.1` | Public | Outside all private ranges — this is Cloudflare's public DNS server |
| `192.168.100.1` | Private | Falls in the 192.168.0.0 – 192.168.255.255 private range |

Private IPs are used inside home and office networks and are not visible on the internet. Public IPs are globally unique and assigned by ISPs — they are visible on the internet.
